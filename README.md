## Answers

### 1. Consequences of exposing an API key
Exposing an API key on GitHub can lead to unauthorized usage, financial loss due to misuse, and potential service disruption.

### 2. Why logging city names is prohibited
City names are considered personal data under privacy laws like GDPR. Logging them can violate user privacy and data protection policies.
